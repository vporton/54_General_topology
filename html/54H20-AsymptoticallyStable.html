asymptotically stable


Let (X,d) be a metric space and f:X→X a continuous function. A point x∈X is said to be Lyapunov stable if for each ϵ>0 there is δ>0 such that for all n∈ℕ and all y∈X such that d⁢(x,y)<δ, we have d⁢(fn⁢(x),fn⁢(y))<ϵ.

We say that x is asymptotically stable if it belongs to the interior of its stable set, i.e. if there is δ>0 such that limn→∞⁡d⁢(fn⁢(x),fn⁢(y))=0 whenever d⁢(x,y)<δ.

In a similar way, if φ:X×ℝ→X is a flow, a point x∈X is said to be Lyapunov stable if for each ϵ>0 there is δ>0 such that, whenever d⁢(x,y)<δ, we have d⁢(φ⁢(x,t),φ⁢(y,t))<ϵ for each t≥0; and x is called asymptotically stable if there is a neighborhood U of x such that limt→∞⁡d⁢(φ⁢(x,t),φ⁢(y,t))=0 for each y∈U.
