a space is T1 if and only if distinct points are separated


Theorem 1.

Let X be a topological space. Then X is a T1-space if and only if sets {x}, {y} are separated for all distinct x,y∈X.

Proof.

Suppose X is a T1-space. Then every singleton is closed and if x,y∈X are distinct, then

{x}∩{y}¯ = {x}∩{y}=∅,
{x}¯∩{y} = {x}∩{y}=∅,

and {x}, {y} are separated. On the other hand, suppose that {x}∩{y}¯=∅ for all x≠y. It follows that {y}¯={y}, so {y} is closed and X is a T1-space. ∎
