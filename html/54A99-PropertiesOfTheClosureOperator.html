properties of the closure operator


Suppose X is a topological space, and let A¯ be the closure of A in X. Then the following properties hold:

  1. 1.

    A¯=A∪A′ where A′ is the derived set of A.

  2. 2.

    A⊆A¯, and A=A¯ if and only if A is closed

  3. 3.

    A¯=∅ if and only if A=∅.

  4. 4.

    If Y is another topological space, then f:X→Y is a continuous map, if and only if f⁢(A¯)⊆f⁢(A)¯ for all A⊆X. If f is also a homeomorphism, then f⁢(A¯)=f⁢(A)¯.
