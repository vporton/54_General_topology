types of limit points


Let X be a topological space and A⊂X be a subset.

A point x∈X is an ω-accumulation point of A if every open set in X that contains x also contains infinitely many points of A.

A point x∈X is a condensation point of A if every open set in X that contains x also contains uncountably many points of A.

If X is in addition a metric space, then a cluster point of a sequence {xn} is a point x∈X such that every ϵ>0, there are infinitely many point xn such that d⁢(x,xn)<ϵ.

These are all clearly examples of limit points.
