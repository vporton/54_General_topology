uniform base


Let X be a Hausdorff topological space. A basis for X is said to be a uniform base if for all xâˆˆX and every neighborhood U of x, only a finite number of the basis sets containing x intersect the complement of U.

For example, in any metric space, the open balls of radius 1n form a uniform base of X.

Any uniform base of X is a point countable base.
